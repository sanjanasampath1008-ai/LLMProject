## Description
This project demonstrates the basic concept of the attention mechanism used in Large Language Models (LLMs). It includes tokenization, word embeddings, attention weights, and the calculation of the final attention output using Python.

## Features
- Tokenization
- Word Embedding
- Attention Weights
- Attention Output Calculation
- Simple Python Implementation

## Technologies Used
- Python

## Files
- `LLMinput.py` – Main program
- `LLM_output.py` – Output program 
## Output
The program displays:
- Tokenized words
- Embedding vectors
- Attention weights
- Final attention output
